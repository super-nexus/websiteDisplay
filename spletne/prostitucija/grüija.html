V Antični Grčiji so moški odkrito živeli razuzdano življenje. Moški so odkrito zahajali k prostitutkam, ki so jih imeli za zabavo.
Prostitutke višjega stanu so se imenovale hetere in so uživale spoštovanje in ugled, ne da bi jim bilo treba skrivati svoj poklic. Družile so se z najbogatejšimi, najbolj izobraženimi in najmočnejšimi možmi svojega časa. Hetere so bile lepotice, izobraženega in uglajenega vedenja, ki so v vseh pogledih prekašale grške žene.
Podoben sloves kot herete so imele tudi auletride (piščalkarice), ki so jih Grki najemali za zabave, na katerih so jim nudile še druge vrste zabave, kot so samo ples in igranje na flavto.
Navadni meščani pa so se morali zadovoljiti z navadnimi prostitutkami, do katerih so se obnašali drugače kot do heter.
V Grčiji so odprli tudi prve javne hiše. Tu so si grki lahko za majhno plačilo kupili spolne usluge. V njih so živele sužnje, ki so za svoje usluge dobivale le hrano in obleke. Njihov zaslužek je pobirala država. Državna blagajna je iz javnih hiš tako dobila velike količine denarja. Prostitutke najnižjega stanu pa so iskale stranke, kar na ulicah in trgih.
